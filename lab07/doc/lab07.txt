	Лабораторна робота №7. Функції
	Ворона Данило Олексійович, КН-923в

1. Переробив 5 і 6 лабораторні роботи так, щоб їх алгоритм викликали функції

2. На перших строках визначили 2 функції, написали їх тіла знизу, а в головному тілі викликали їх
використавши вибір юзера: 1 - функція для перетворення числа в слова, 2 - знайти НСД

3. У тілах функцій замість вводу користувача, використали рандом, обмежений 0 і 9999.

4. Відладчик: Зупинивши програму в тілі мейн(22 строка), існує тільки 1 змінна function,
яка набуває значень 1 або 2. Тільки після того як брейкпоінт буде поставлений у тіло однієї з функцій
можна побачити її змінні, згенеровані рандомом:

1) Для першої 
(int) input = 8362
(int) num_pos = 4
після циклу
(int) input = 0
(int) num_pos = 0

2) Для другої
(int) number_one = 5033
(int) number_two = 9639
(int) NSD = 0
після циклу
(int) number_one = 5
(int) number_two = 5
(int) NSD = 5
